c++ lab
=======
Idiot-proof steps to making programs with C++.

**It's magic!**
* Chapter 1: functions

    ```g++ -Wall -o $name $name.cpp```

* Chapter 2: array

    ```g++ -Wall -o $name $name.cpp```

* Chapter 3: strings

    ```g++ -Wall -std=c++11 -lboost_regex -o $name $name.cpp``` or ```g++ -Wall -std=c++11 -o $name $name.cpp```


* Chapter 4: struct

    ```g++ -Wall -o $name $name.cpp```
